# SEOギャップ分析 — 競合との差分を特定

対象キーワードで上位表示されている競合ページと、Lオペ LP（l-ope.jp/lp）のコンテンツギャップを分析してください。

## 引数
- $ARGUMENTS: 分析対象キーワード（例: "クリニック LINE"）

## 分析手順

### 1. 自社LPのコンテンツ分析
- `app/lp/components/` の全コンポーネントを読み、LPの全コンテンツを把握
- `.claude/lp-content.md` のLP原稿を確認
- h1, h2, h3の見出しテキストを全て抽出
- 使用されているキーワードの出現頻度を集計

### 2. キーワード対象ページの構造分析
- 指定キーワードに関連するセクションがLP内のどこにあるか特定
- そのセクションの見出し・本文でキーワードがどう使われているか分析
- 内部リンク構造の確認

### 3. コンテンツギャップの特定
以下の観点でLPに不足している要素を特定:
- **見出しの網羅性**: 指定キーワードのサブトピックがh2/h3に含まれているか
- **説明の深さ**: 各機能の説明が十分か、ユーザーの疑問に答えているか
- **信頼性要素**: 導入事例、数値実績、第三者評価の有無
- **CTA最適化**: 適切な位置にCTAが配置されているか
- **FAQ充実度**: 指定キーワードに関連するFAQが網羅されているか

### 4. 競合比較ポイント
以下の競合サービスとの差別化ポイントを分析:
- Lステップ（LINE公式アカウント拡張ツール）
- Liny（LINE運用ツール）
- その他クリニック向けLINEツール
