## 3. サービス概要（ABOUT）

### Lオペ for CLINIC とは

LINE公式アカウントを「クリニックの業務基盤」へ進化させる、
医療機関専用のオールインワン運用プラットフォームです。

| コア価値 | 説明 |
|----------|------|
| **LINE起点のCRM** | 友だち追加から問診・予約・フォローアップまで、患者とのすべてのコミュニケーションをLINE上で一元管理。 |
| **業務オールインワン** | 予約管理・会計・配送・カルテ連携・スケジュール管理まで、バラバラだったツールを1つに統合。 |
| **ノーコード自動化** | リッチメニュー・フォーム・自動アクションをGUIで構築。エンジニア不要で現場スタッフが運用可能。 |

## 4. 機能紹介（FEATURES）

### クリニック運営に必要なすべての機能を搭載

汎用LINEツールにはない、医療現場に最適化された40以上の機能群。
AI自動返信・AIカルテ・AIメニュー生成など、AI機能も標準搭載。
日常のオペレーションからマーケティングまでカバーします。

---

### 4-1. 患者CRM

#### タグ管理
患者にカラー付きタグを自由に設定。「初診」「美容」「ダイエット」など、診療内容・属性ごとに分類し、セグメント配信の基盤を構築します。友だち追加時の自動タグ付けにも対応。

> 10色のカラー / 自動タグルール / 一括タグ付け

#### 対応マーク（ステータス管理）
「未対応」「対応中」「完了」「重要」「保留」など、患者ごとの対応状況を色付きマークで可視化。スタッフ間の引き継ぎ漏れを防止し、対応品質を標準化します。

> 6段階のステータス / カスタムラベル / 一覧での色分け表示

#### 友だち情報欄（カスタムフィールド）
患者ごとに自由な情報フィールドを定義・蓄積。「前回処方薬」「アレルギー」「希望メニュー」など、クリニック独自の管理項目をLINE CRMに統合できます。

> フィールド自由定義 / Webhook連動で自動更新

#### 患者重複排除・統合
重複患者を自動検出し、ワンクリックでマージ処理を実行。名前・電話番号・LINE IDなどの類似度スコアリングでデータの整合性を維持します。

> 自動検出 / 類似度スコア / ワンクリックマージ

---

### 4-2. メッセージ配信

#### セグメント配信
タグ・マーク・友だち情報の組み合わせで、配信対象を精密に絞り込み。「再診促進は3ヶ月未受診の美容タグ患者のみ」といった高度なターゲティングが可能です。

> 複数条件の AND/OR / 除外条件 / 配信前プレビュー（対象者数・一覧）

#### テンプレート管理
よく使うメッセージをテンプレート化してカテゴリ別に整理。変数（{name}・{patient_id}）の自動挿入で、パーソナライズされたメッセージを瞬時に送信できます。

> テキスト＆画像テンプレ / カテゴリフォルダ / 変数置換

#### 予約リマインド
予約日前に自動でLINEリマインドを送信。無断キャンセルの削減と来院率の向上を実現。送信結果（成功・失敗・UID無し）もリアルタイムで追跡できます。

> 個別 / 一括リマインド / CSV出力 / 送信ステータス管理

#### A/Bテスト
メッセージの複数バリアントを自動分割配信し、開封率・CV率を比較検証。データドリブンで配信内容を最適化できます。

> バリアント自動分割 / 成功率比較 / 結果の可視化

---

### 4-3. ノーコード構築

#### リッチメニュービルダー
ドラッグ操作でLINEリッチメニューのボタン配置を自由に設計。各ボタンにURL・電話・メッセージ送信・テンプレート送信・タグ操作・メニュー切替など複雑なアクションを設定できます。

> ビジュアルエディタ / 6分割〜自由配置 / 条件分岐アクション / LINE API自動連携

#### Flex Messageビルダー
LINE Flex Message（カード型リッチメッセージ）をノーコードで作成。カルーセル・ボタン・画像をプリセットから選んでビジュアルエディタで編集、そのまま配信できます。

> プリセットテンプレート / ウィザード形式 / カルーセル・パネル対応

#### 回答フォームビルダー
問診票・アンケート・同意書などのフォームをGUIで作成。患者はLINE上からワンタップでフォームにアクセスし回答。回答データは管理画面に自動集約されます。

> フォルダ整理 / 公開URL自動生成 / ファイルアップロード対応 / 回答一覧閲覧

#### チャットボットビルダー
会話フロー（シナリオ）をビジュアルエディタで設計。メッセージ・質問・アクション・条件分岐ノードを組み合わせて、患者の応答に応じた自動対話を構築できます。

> ビジュアルフローエディタ / 質問・条件分岐・アクションノード / シナリオテンプレート

#### アクション自動化
「友だち追加→挨拶メッセージ→タグ付与→リッチメニュー切替」のような一連のワークフローをステップ形式で構築。条件分岐にも対応し、患者の状態に応じた自動対応を実現します。

> ステップ実行 / 条件分岐（タグ・マーク・日時）/ 遅延送信（1分〜1日）

### 4-4. 業務管理

#### 予約・スケジュール管理
月別・週別の予約カレンダーに加え、複数医師の並列スケジュール管理・医師別フィルタ・休日設定・日付オーバーライドに対応。患者のLINE経由予約をそのままダッシュボードで管理できます。

> 月/週ビュー / 複数医師並列予約 / 医師別フィルタ / 休日設定 / 予約ステータス追跡

#### キャンセル待ち
予約キャンセルが発生すると、待ちリストの患者へLINEで自動通知。空き枠を無駄なく活用し、予約充足率を向上させます。

> 待ちリスト自動管理 / キャンセル時LINE自動通知 / 先着順マッチング

#### 会計・決済管理
Square/GMO決済連携による売上追跡・銀行振込の消込処理・返金管理まで、クリニックの金流をダッシュボードで一元可視化。申し込みからGMO決済までLINE経由でシームレスに完結。

> Square/GMO連携 / 振込消込 / 返金処理 / 日次KPIレポート


#### 配送・発送管理
オンライン処方の配送業務をシステム化。ヤマトB2形式のCSV出力・追跡番号管理・配送ラベル印刷まで対応。処方薬の発送漏れを防止し、患者へ追跡リンクも自動共有できます。

> ヤマトB2 CSV出力 / 追跡番号一括更新 / 配送ステータス管理

### 4-5. AI機能

#### AI自動返信
AIが患者からの問い合わせ内容を理解し、クリニックのFAQ・処方情報を踏まえた返信文を自動生成。スタッフ確認後に送信するフローで品質を担保しつつ、対応時間を最大80%短縮。自動返信モードへの切り替えも可能で、営業時間外の自動送信にも対応。

> AI返信候補自動生成 / 修正指示→再生成 / 自動送信モード / AI返信統計

#### AI自動学習（RAG方式）
スタッフの修正送信・手動返信をAIが自動学習。embeddingベクトルで類似事例を蓄積し、使うほど返信精度が向上します。

> 修正・手動返信からの自動学習 / 類似事例参照 / 学習例の管理画面

#### 音声カルテ自動生成
診察中の会話をワンタップ録音。AIが音声を文字起こしし、SOAP形式のカルテを自動生成。薬剤名・症状名の自動抽出で、カルテ作成時間を1件あたり数分に短縮。

> SOAP自動生成 / 医療用語自動抽出 / カルテテンプレート連携

#### AIモデル切替
用途やコストに応じて、管理画面からAIモデルを自由に選択。高精度モデルと高速モデルを使い分けて、最適なバランスで運用できます。

> 管理画面から即切替 / コスト・精度の最適化

### 4-6. 在庫・商品管理

#### 在庫管理・在庫台帳
処方薬・消耗品の在庫をリアルタイムで把握。入出庫の自動記録、在庫台帳で推移を可視化。発注判断に必要な情報をワンストップで提供。

> 入出庫自動記録 / 在庫推移グラフ / 要発注アラート

#### 商品マスタ管理
処方薬・施術メニュー・消耗品の商品情報をまとめて管理。価格・在庫連動・決済連携設定まで一元管理。

> 商品・価格・カテゴリ一括管理 / 在庫・決済連動 / 商品別売上分析

#### ポイント自動付与
購入金額・初回購入・累計額に応じたポイントルールを設定。決済Webhook連動でポイントを自動付与し、リピート促進を自動化します。

> 3種ルール（購入毎/初回/累計閾値）/ 決済Webhook連動 / マイページ表示

> 処方X日後に自動送信 / 処方薬別テンプレート / 再診促進エスカレーション

### 4-8. 分析・レポート

#### リアルタイムダッシュボード
予約数・売上・新規/リピート比率・LINE友だち推移・決済率・来院率など、クリニック経営に必要なKPIをリアルタイムで一覧表示。13項目のKPIカードをドラッグ&ドロップで自由にカスタマイズ。日別・週別・月別・カスタム期間での比較分析が可能です。

> D&DカスタムKPI / 商品別売上 / コンバージョンファネル / 期間比較

#### 流入経路トラッキング
LINE友だち追加の流入経路をUTM別に計測。広告・SNS・店頭QRなど、どの経路からの友だち追加が多いかをデータで可視化。日別トレンド・コンバージョン分析で広告効果を最適化できます。

> UTM計測 / Cookie+リダイレクト方式 / 日別トレンド / CV分析

#### メッセージ配信ログ
全メッセージの送受信履歴を一元管理。個別・一斉・リマインド・ステップ配信など種類別に検索可能。送信成功/失敗/UID無しのステータスで配信品質を監視できます。

> 方向別ログ / ステータス追跡 / ページネーション / エラー詳細表示

#### メディア・アセット管理
リッチメニュー画像・テンプレート画像・PDFなどのメディアファイルをフォルダ管理。各機能から横断的に利用でき、クリエイティブ資産を効率的に運用できます。

> フォルダ整理 / 画像・PDF対応 / リッチメニュー・テンプレ連携

## 5. 差別化・強み（STRENGTHS）

### Lオペ for CLINIC が選ばれる4つの理由

院長と事務スタッフ1人だけで、カルテ・予約・LINE配信・CRM・決済・配送まですべて回せる。
複数ツールの契約も追加人員も不要にする「少人数クリニック起点」の設計思想が最大の差別化です。

---

### 01. 院長＋事務1人で全業務完結

- カルテ・予約・LINE配信・CRM・決済・配送…すべてが1つの管理画面に集約。事務スタッフ1人ですべて回せる
- 自動リマインド・自動フォローアップ・AI自動返信で、手作業の連絡業務をほぼゼロに
- ITに詳しくないスタッフでもノーコードで操作完結。エンジニア不要、外注費ゼロ

### 02. ツール代＋人件費まるごと削減

- カルテ（月5〜10万）＋予約システム（月3〜5万）＋LINE配信/CRM（月5〜10万）…個別導入で月額15〜30万。Lオペなら月15〜20万でオールインワン
- 複数ツールの管理に必要だった人員が不要に。人件費も含めたトータルコストで大幅削減
- 1つのツールを覚えるだけ。スタッフの学習コスト・ツール間の二重入力もゼロ

### 03. クリニック専用設計（vs. Lステップ・Liny等の汎用ツール）

- Lステップ・Linyなどの汎用LINE配信ツールは飲食・EC向け。医療特有の「問診→予約→診療→処方→フォロー」導線に未対応
- 患者CRM・対応マーク・処方管理・配送追跡まで、クリニック業務フローに完全特化
- 友だち追加→自動問診→予約→来院→決済→配送→フォローまで1画面で追跡。データ分断ゼロ

### 04. すべてがLINEで自動化

- 予約確定・リマインド・決済完了・発送通知・追跡番号…すべてのステータス変化がLINEに自動通知。電話・メールの取りこぼしゼロ
- 患者はLINEを見るだけで自分の状況を把握。問い合わせ電話が激減し、受付業務を大幅削減
- リッチメニュー・配信・自動アクションもGUIで即設定。管理画面から院長自身がすぐ反映

## 7. 導入の流れ（FLOW）

お問い合わせから最短2週間で運用開始。初期設定はサポートチームが代行するため、現場の負担はほぼゼロです。

| STEP | タイトル | 内容 |
|------|----------|------|
| 01 | お問い合わせ | フォームまたはLINEからお気軽にご相談ください。デモ画面をお見せしながら、貴院の課題をヒアリングします。 |
