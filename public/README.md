# メディカルダイエットクリニック

マンジャロ（GLP-1医薬品）のオンライン処方プラットフォーム

## 📋 プロジェクト概要

このプロジェクトは、マンジャロ（チルゼパチド）などのGLP-1受容体作動薬によるメディカルダイエットのオンライン診療・処方プラットフォームです。患者様がオンラインで診察予約を行い、医師の診察を経て適切な医薬品の処方を受けられるサービスを提供します。

### 主な特徴

- 🏥 **オンライン診療**: 初診から全国どこでもオンラインで受診可能
- 💊 **マンジャロ処方**: FDA承認の最新GLP-1医薬品
- 📦 **最短当日発送**: 平日15時までの決済で即日発送
- 💰 **明確な料金体系**: 診察料・送料込みの分かりやすい価格設定
- 📱 **レスポンシブデザイン**: スマートフォン・タブレット・PC対応

## 🔌 API エンドポイント

### 診療予約
```javascript
// 予約一覧取得
GET /tables/reservations?page=1&limit=100

// 予約詳細取得
GET /tables/reservations/{id}

// 予約作成
POST /tables/reservations
Body: {
  name, nameKana, birthday, gender, email, phone,
  weight, height, bmi, medicine, reservationDate,
  reservationTime, medicalHistory, notes
}

// 予約更新
PUT /tables/reservations/{id}

// 予約削除
DELETE /tables/reservations/{id}
```

### 医薬品情報
```javascript
// 医薬品一覧取得
GET /tables/medicines

// 医薬品詳細取得
GET /tables/medicines/{id}
```

## 🚀 使い方

### 患者様向け

1. **ホームページを閲覧**
   - マンジャロについての詳細情報を確認
   - 料金プランを比較
   - FAQで疑問を解決

2. **診療予約**
   - 「診療予約」ボタンをクリック
   - 必要事項を入力（氏名、連絡先、体重・身長など）
   - 希望の薬剤と診察日時を選択
   - 予約を送信

3. **診察・処方**
   - オンラインビデオ通話で医師が診察
   - 適切と判断された場合、処方箋発行
   - クレジットカードで決済

4. **お薬配送**
   - 最短当日発送
   - クール宅急便でご自宅にお届け

### 医療機関管理者向け

1. **予約管理**
   - APIを使用して予約データを取得
   - 予約ステータスの更新（確定、完了、キャンセル）

2. **医薬品在庫管理**
   - 医薬品マスターデータの更新
   - 在庫状況の管理

3. **お問い合わせ対応**
   - お問い合わせデータの確認
   - 対応ステータスの更新

## ⚠️ 注意事項

### 医療に関する重要事項

1. **処方の可否**: 医師の診察により処方が適さないと判断される場合があります
2. **副作用**: 吐き気、下痢、便秘などの消化器症状が現れる可能性があります
3. **禁忌**: 1型糖尿病、妊娠中・授乳中の方は使用できません
4. **保険適用**: ダイエット目的での処方は自由診療（保険適用外）です

### 技術的制約

- このサイトは静的Webサイトであり、実際の決済機能は実装されていません
- ビデオ通話機能は外部サービスとの連携が必要です
- 実際の医療機関として運用する場合は、医療法に基づく許可が必要です
