利用規約

この利用規約（以下「本規約」）は、のなめビューティー（以下「当院」）が提供するオンライン診療サービス（以下「本サービス」）の利用条件を定めるものです。本サービスをご利用いただく際には、本規約に同意いただいたものとみなします。

第1条（適用）

  1. 本規約は、本サービスの利用に関する当院と利用者との間の権利義務関係を定めることを目的とし、利用者と当院との間の本サービスの利用に関わる一切の関係に適用されます。
  2. 当院が本サービス上で掲載する諸注意等は、本規約の一部を構成するものとします。

第2条（利用資格）

本サービスは、以下の条件を満たす方がご利用いただけます：

  1. 18歳以上の方
  2. 本規約に同意いただける方
  3. 当院の医師が診察の上、処方が適切と判断した方
  4. 日本国内に配送先住所を有する方

第3条（サービス内容）

本サービスは、オンライン診療により医薬品の処方を受けられるサービスです。診察は当院の医師が行い、医師が適切と判断した場合のみ処方いたします。

第4条（予約及び診察）

  1. 利用者は、当院所定の方法により診察の予約を行うものとします。
  2. 診察は、通話またはビデオ通話により行います。
  3. 事前に問診票への回答が必要です。
  4. 診察時間は、初診15-20分程度、再診10分程度です。
  5. 医師の判断により、処方をお断りする場合があります。

第5条（料金及び支払い）

  1. 本サービスの料金は、当院ウェブサイトに記載された価格とします。
  2. 料金には、診察料、医薬品代、送料が含まれます。
  3. 支払いは、クレジットカードまたは銀行振込により行うものとします。
  4. 決済完了後のキャンセルは、発送前に限り可能です。

第6条（配送）

  1. 医薬品は、ヤマト運輸（クール便チルド）にて配送します。
  2. 平日12時までの決済完了で最短当日発送いたします。
  3. 配送先は、日本国内に限ります。
  4. 配送状況の確認は、追跡番号により可能です。

第7条（禁止事項）

利用者は、本サービスの利用にあたり、以下の行為を行ってはなりません：

  1. 虚偽の情報を提供する行為
  2. 他人になりすます行為
  3. 処方された医薬品を転売または譲渡する行為
  4. 当院または第三者の権利を侵害する行為
  5. 法令に違反する行為
  6. 本サービスの運営を妨害する行為
  7. その他、当院が不適切と判断する行為

第8条（利用停止）

当院は、利用者が以下のいずれかに該当する場合、事前の通知なく本サービスの利用を停止することができます：

  1. 本規約に違反した場合
  2. 提供された情報に虚偽があった場合
  3. 支払期日を過ぎても料金の支払いがない場合
  4. その他、当院が本サービスの利用を適当でないと判断した場合

第9条（免責事項）

  1. 当院は、医薬品の効果・効能について個人差があることを説明します。
  2. 医薬品の使用による副作用については、医師の指示に従い適切に対応してください。
  3. 天災地変、通信障害等、当院の責めに帰すべからざる事由による本サービスの中断・遅延については、当院は責任を負いません。

第10条（個人情報の取扱い）

当院は、利用者の個人情報を、プライバシーポリシーに基づき適切に取り扱います。

第11条（規約の変更）

当院は、必要に応じて本規約を変更することができます。変更後の規約は、当院ウェブサイトに掲載した時点から効力を生じるものとします。

第12条（準拠法及び管轄裁判所）

  1. 本規約の解釈にあたっては、日本法を準拠法とします。
  2. 本サービスに関して紛争が生じた場合には、東京地方裁判所を第一審の専属的合意管轄裁判所とします。
